Jest

Jest é uma lib de testes javascript, também utilizada com react native, react e outros.

Instalar

  • npm install --save-dev jest
    •

    Testes

    Forma

  • Os testes se fazem num arquivo __nome.test.ts ou __nome.test.js
    •

    Forma de testar

    it

  • it -> declara uma função de teste. Com 2 parâmetros.
  • O primeiro parâmetro recebe um texto descrevendo o teste
  • Segundo parâmetro recebe uma função, dentro dela receberá o teste

    • expect toBe toEqual

  • expect() -> recebe uma função que retorna algum valor
  • .toBe() -> O valor que a função deve retornar
  • .toEqual() -> Quando comparamos arrays ou objetos, é necessário usar .toEqual(), pois em javascript {} === {} retorna false

    • describe

  • describe é como se agrupa varios testes em um só lugar
    •

    Async Functions

  • É praticamente a mesma coisa, porém temos que usar async await ou .then
    •

    Table

    COMANDO Função
    it("", () => { }) Cria um novo Teste
    expect(valor) Determina qual valor sera avaliado
    .toBe(valor) Determina o que deve resultar o expect
    .toEqual(valor) Determina o que deve resultar o expect, porém nesse caso objetos e arrays podem ser igualados
    .toBeTruthy() Determina que deve resultar true
    .toBeFalsy() Determina que deve resultar false
    .toBeUndefined() Determina que deve resultar undefined
    .toBeNull() Determina que deve resultar null
    .toBeGreaterThan(numero) Determina que deve resultar maior que este número
    .toBeGreaterThanOrEqual(numero) Determina que deve resultar maior ou igual que este número
    .toBeLessThan(numero) Determina que deve ser menor que este número
    .toBeLessThanOrEqual(numero) Determina que deve ser menor que este número
    .toBeCloseTo(numero) Determina que deve ser aproximadamente este valor
    .toMatch(/RegEx/) Determina que deve ser compatível com a RegEx, podemos comparar strings.
    .toContain(valor) Determina que um array deve conter este valor, seja string, seja number
    .not Determina que o próximo deve ser negativo: expect(null).not.toBeTruthy()
    .toThrow(error) Determina que deve retornar um Throw new Error()
