    • Introdução a Realm DB

    O Realm Database é um banco de dados orientado a objetos para plataformas móveis.

  • Realm -> O banco de Dados
  • Class -> A Tabela
  • Objetos -> Documento
  • Propriedades -> Campos
    •

    Estrutura

  • -- databases
  • ---- schemas
  • ------ UserSchema.tsx -> importar tipo { ObjectSchema}, export default Scheme: ObjectSchema = {name: 'nome', properties: {_id: 'int', name: 'string'}, primaryKey: '_id'}
  • -- realm.ts -> Declara-se getRealm(): export default const getRealm = async () => { Realm.open({path: 'nome', schema: []})}
    •

    Comandos

    COMANDO Função
    realm.write(() => {}) Diz ao banco que enviaremos algo para ele
    realm.create("schema", {propriedades}) É a forma de criar um novo objeto na tabela. Caso passemos como terceiro argumento Realm.UpdateMode.Modified ele sobreescreve, não cria
    realm.objects<typescript>('nome_tabela') Retorna os objetos do banco

    Tipos

    Primitivos

  • int -> numero inteiro
  • string -> texto
  • float -> numero com vírgula
  • double -> número com vírgula mais preciso
  • data -> dados em binário
  • date -> data, dia, mes, ano
  • object
  • objectId
  • mixed -> qualquer tipo de dado realm
  • decimal128 -> numero decimal de alta precisão
  • list -> array
  • [] -> coloca-se depois de qualquer primitivo para indicar que é um array
