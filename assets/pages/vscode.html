Atalhos VS Code Windows

Atalho Função
ctrl + s Salvar
ctrl + alt + s Salvar todas as abas abertas
ctrl + z Desfazer última ação
ctr l + shift + z Refazer ação desfeita
ctrl + n Criar arquivo
ctrl + o Abrir arquivo
ctrl + k ctrl + o Abrir diretório
ctrl + b Ativar/Desativar barra lateral
ctrl + f Barra de busca
ctrl + h Substituir palavra. Por padrão é trocada apenas uma palavra.
ctrl + alt + enter faz com que troque todas as palavras.
ctrl + ; Comenta/Descomenta linha inteira.
Caso seja um bloco selecioinado, comenta o bloco inteiro
shift + alt + ↓/↓ Duplicar linha
alt + ↑/↓ Move linha para cima/baixo
ctrl + d Seleciona em conjuto a próxima ocorrência da mesma palavra do código
Caso acabe as ocorrências, ele começa a procurar desde o início da página
ctrl + l Selecionar linha
alt + z Ativar/Desativar word wrap
alt + numero Navegar entre abas
ctrl + tab Navegar entre abas
ctrl + PgDn/PgUp Navegar entre abas
ctrl + g Acessar linha
ctrl + shift + l Selecionar todas as instâncias da palavra atual
ctrl + w Fechar guia atual
ctrl + ' Abrir terminal
alt + shift + f Formatar documento
alt + l alt + o Abrir Live Server
