HTML5

HyperText Markup Language

O HTML (HyperText Markup Language) é a linguagem de marcação usada para estruturar o conteúdo das páginas da web, definindo elementos como texto, imagens e links. É a base da World Wide Web, permitindo que os navegadores interpretem e exibam o conteúdo das páginas de maneira formatada.

CSS3

Cascading Style Sheets

O CSS (Cascading Style Sheets) é uma linguagem de estilo utilizada na criação de páginas da web. Ele permite que os desenvolvedores web definam a aparência e o layout de elementos HTML, controlando propriedades como cores, fontes, tamanhos e posicionamento. Ao separar o conteúdo (HTML) da apresentação (CSS), o CSS facilita a criação de páginas visualmente atraentes e consistentes em toda a web, tornando-o uma parte fundamental do desenvolvimento web moderno.

JavaScript

JavaScript

O JavaScript é uma linguagem de programação amplamente utilizada para tornar as páginas da web interativas e dinâmicas. Com o JavaScript, os desenvolvedores web podem adicionar funcionalidades avançadas, como animações, validação de formulários, manipulação de elementos da página e muito mais. Ele é executado diretamente no navegador do usuário, permitindo que as páginas reajam a ações do usuário em tempo real, tornando a experiência na web mais envolvente e responsiva. O JavaScript é uma pedra angular do desenvolvimento web moderno, permitindo a criação de aplicativos web ricos e interativos.

Markdown

Markdown Language

Markdown é uma linguagem de marcação leve amplamente usada para formatar texto simples em documentos formatados de maneira estruturada. Ela permite que os escritores adicionem facilmente elementos como cabeçalhos, listas, links e formatação de texto, usando uma sintaxe simples e intuitiva. Markdown é especialmente popular em plataformas de blogging, wikis e sistemas de gerenciamento de conteúdo, pois facilita a criação rápida e legível de conteúdo online sem a necessidade de conhecimento técnico avançado. É amplamente adotado por sua simplicidade e eficiência na criação de documentos bem formatados e é convertido facilmente em HTML ou outros formatos de saída por meio de ferramentas de conversão.

MySQL

My Structured Query Language

O MySQL é um sistema de gerenciamento de banco de dados relacional (RDBMS) amplamente utilizado que permite o armazenamento, recuperação e gerenciamento eficiente de dados. Ele utiliza uma linguagem de consulta chamada SQL (Structured Query Language) para interagir com o banco de dados. Com o MySQL, os desenvolvedores e administradores de banco de dados podem criar bancos de dados, tabelas e definir relacionamentos entre os dados. Ele oferece recursos robustos de segurança, escalabilidade e desempenho, tornando-o uma escolha popular para aplicativos web, sites e uma variedade de outras aplicações de software que exigem armazenamento e recuperação de dados. O MySQL é um software de código aberto amplamente adotado por sua confiabilidade e flexibilidade, e é suportado por uma comunidade ativa de desenvolvedores e uma variedade de ferramentas e recursos para gerenciamento e desenvolvimento de bancos de dados.

VS Code

Visual Studio Code

O Visual Studio Code, frequentemente abreviado como VS Code, é um ambiente de desenvolvimento integrado (IDE) altamente popular e altamente extensível, amplamente adotado por programadores em todo o mundo. Graças à sua simplicidade e versatilidade, o VS Code é conhecido por oferecer uma experiência de codificação eficiente e altamente personalizável, atendendo a uma variedade de linguagens de programação e necessidades de desenvolvimento.
