Node.js

O que é

O Node.js é um Runtime de JavaScript. Um Runtimee é um ambiente de execução no qual um código é executado. Node.js usa o motor do Google Chrome, V8, para rodar códigos javascript fora do navegador.

Node.js Basics

Para Roda um arquivo no node, basta escrevê-lo no terminal.

Também poedemos escolher arquivos para o node rodar, entrado com o termina no diretório do arquivo e digitando a palavra node e o nome do arquivo que se quer utilizar. Escrever o .js no final do nome do arquivo é opcional.

O Node.js não roda no navegador, então o console deixa de ser o console do navegador e passa ser a janela do terminal.

No navegador, a palavra window se refere ao objeto window. Em Node, a variável chamada global, é uma palavra chave do node para se referir ao objeto global. Podemos verificar seus métodos com console.log(global), que são é muito menor que o window

O Node.js inclui dentro de si várias bibliotecas e módulos integrados que dão acessa à funcionalidades específicas do sistema. No JavaScript vanilla usaria-se imports, porém em node.js ao invés de import, usa-se o comando require('')

Node require ex: 
    
    const os = require('os') || os é um módulo do node que fornece infomações do sistema, ele possui alguns metodos como type(), version() e homedir()

    console.log(os.type())
    console.log(os.version())
    console.log(homedir())

Módulos Node.js

Existem muitos módulos nativos do node.js que podemos importá-los, porém também podemos criar nossos próprios módulos e importar módulos de outros desenvolvedores.

Para exportar as funções de um módulo usa-se o comando module.exports e passa as funções dentro de um objeto

ex: 

    || math.js Archive
    const add = (a, b) => a + b;
    const subtract = (a, b) => a - b;
    const multiply = (a, b) => a * b;
    const divide = (a, b) => a / b;
    
    module.exports = {add, subtract, multiply, divide}

    ||other Archive
    const math = require('.math') 

    console.log(math.add(2,5))

Podemos também desestruturarr essa meneira de importar

ex: 

    const { add, subtract, multiply, divide } = require('./math')

    console.log(add(3, 9))

Podemos desestruturar o módulo que criamos também eliminando o const e importando diretamente com o comando inline exports. sem necessidade de usar o module.exports

ex: 

    exports.add = (a, b) => a + b;
    exports.subtract = (a, b) => a - b;
    exports.multiply = (a, b) => a * b;
    exports.divide = (a, b) => a / b;

O node.js não tem algumas API's nativas do js, como por exemplo Fetch

Todos os módulos tem que ser importados antes de serem chamados, declarando-o como uma variável e usando require('')

ex: const os = require('os')

path

Módulo path contém métodos com caminhos de diretórios.

path methods

path methods Função Exemplo
.dirname() Retorna o nome do diretório. console.log(path.dirname(__filename))
.basename() Retora unicamente o nome do arquivo, sem o caminho console.log(path.basename(__filename))
.extname() Retorna unicamente a exetensão do arquivo. (letras após o . ) console.log(path.extname(__filename))
.parse() Retorna um ojeto que mostra a root, a dir, base, ext e name console.log(path.parse(__filename))
.join(__dirname , ' ', ' ') É um jeito melhor de direcionar caminhos
Entre as aspas vai o nome das pastas até chegar ao arquivo que queremos		 path.join(__dirname, 'files', 'texts', 'hello_world.txt')
